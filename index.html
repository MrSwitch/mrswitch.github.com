Wireframes

I like creating mashups to highlight a technology. These are like mini-apps, a little unpolished but fun to build

  • Graffiti - Image manipulation An example of the FileAPI to select an image and read it into a Canvas. Draw a doodle, then download it using a FileSaver.
  • WebRTC Demo - This uses getUserMedia and PeerConnection to capture and stream video between two clients. Its built using NodeJs with socket.io to connect two or more clients up.
  • SplashySplashy - Video chat and Hello.js demo - I worked with a internet video communications company called TokBox in the first half of 2012. This was a demo i built.

  • SVG StickMan inspired by DrawAStickMan

Scripts

  • Hello.js - An extensible SDK for OAuth 2.0 client side authentication and REST calls to service providers API's like Facebook Graph, Windows Live, Google +... etc.
  • notification.js - Shims the Notification API with the best alternatives.
  • dropfile.js - Shims up IE<10 to support the FileReader via a SilverLight plugin
  • jquery-seadragon.js - Microsoft Seadragon is a large image zoomer, and this script enables there script to work on touch devices
